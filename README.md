# DISCRET — Sistema de Reservas para Motel

Sistema web de administración y reservas desarrollado para gestión de habitaciones, clientes y reservas en tiempo real.

## Características

- Panel administrativo moderno
- Gestión de habitaciones
- Gestión de reservas
- Confirmación de reservas
- Agenda diaria
- Clientes y reportes
- Sidebar reutilizable
- Diseño responsive
- Interfaz dark premium

---

## Tecnologías utilizadas

- React
- TypeScript
- Vite
- React Router DOM
- Lucide React
- CSS3

---

## Instalación del proyecto

Clonar repositorio:

```bash
git clone https://github.com/Latasoft-Organizacion/discret-reserva-front.git

Ingresar carpeta del proyecto
cd discret-reserva-front


Instalar dependencias
npm install


Ejecutar proyecto
npm run dev


Estructura del proyecto
src/
 ├── assets/
 ├── components/
 ├── pages/
 ├── styles/
 ├── router/
 └── App.tsx
